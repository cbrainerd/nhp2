# NHP2

## Quickstart

To run the program:

    python -m wgups


## Development

### Unittests

To run unittests:

    pip install -r test-requirements.txt
    make pytest

### Pylint

To run pylint:

    pip install -r test-requirements.txt
    make pylint


### Black

To run Black (code formatting):

    pip install -r test-requirements.txt
    make pylint
